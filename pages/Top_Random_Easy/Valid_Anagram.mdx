# 242. Valid Anagram


- [Leetcode](https://leetcode.com/problems/valid-anagram/description/)


Given two strings `s` and `t`, return true if `t` is an anagram of `s`, and `false` otherwise.

- An Anagram is a word or phrase formed by rearranging the letters of a different word or phrase, typically using all the original letters exactly once.


## Example 1:

```
Input: s = "anagram", t = "nagaram"
Output: true
```

## Example 2:

```
Input: s = "rat", t = "car"
Output: false
```
<details>
  <summary>Hint (Click to expand)</summary>
  Sort the characters in both strings, then compare whether they are equal or not.
</details>

## Solution

<details>
  <summary>Click here</summary>

  ```js
  var isAnagram = function(s, t) {
    if (s.length !== t.length) {
      return false;
    }
    let map = {};
    for (let i = 0; i < s.length; i++) {
      if (map[s[i]]) {
        map[s[i]]++;
      } else {
        map[s[i]] = 1;
      }
    }
    for (let i = 0; i < t.length; i++) {
      if (map[t[i]]) {
        map[t[i]]--;
      } else {
        return false;
      }
    }
    return true;
  };
  ```

  `Space complexity: O(1)` - The function uses a constant amount of additional space (a fixed-size map object) to store character frequencies. The space used does not depend on the input size, so the space complexity is O(1).

`Time complexity: O(n)` - The function uses two for loops, each iterating through the input strings s and t, which have lengths 'n' and perform constant-time operations (mostly dictionary lookups and arithmetic operations) inside the loops. Therefore, the time complexity is O(n).



</details>
